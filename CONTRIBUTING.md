# Contributing

This repository is a **read-only mirror** of the primary repository on
[sourcehut](https://git.sr.ht).

**Do not open pull requests or issues here.** All contributions are managed
through sourcehut:

- **Repository:** https://git.sr.ht/~the-commits/tjenamors-se-docker-cleanup
- **Patches:** send via email to `~the-commits/tjenamors-se-docker-cleanup@lists.sr.ht`
- **Issues:** https://todo.sr.ht/~the-commits/tjenamors-se-docker-cleanup

The GitHub mirror is updated automatically on every push to sourcehut.
